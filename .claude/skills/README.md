# Skills

Learned patterns from retrospectives. Skills are executable knowledge extracted from successful workflows.

## Usage

Skills are loaded on-demand when relevant to current work. Use `/retro` to create new skills from session patterns.

## Skill Template

```markdown
---
name: [skill-name]
trigger: [when to suggest]
source: [retro session or manual]
---

# [Skill Name]

## Context
When to use (1-2 sentences)

## Pattern
Step-by-step execution

## Example
Concrete application

## Anti-Patterns
What NOT to do
```

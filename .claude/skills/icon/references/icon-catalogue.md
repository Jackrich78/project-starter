# Icon Catalogue

Curated Lucide icons for carousel slides. Max 1 per slide, never on hook/CTA.
Render: `node .claude/skills/icon/scripts/render-icon.cjs <name> 120 1.0`

## Constraints
- Max 1 icon per slide
- Never on Slide 1 (Hook) or Slide 6 (CTA)
- 120px at 100% opacity (default)
- 200px at 100% opacity for hero icon (rare)
- Position: top-right of content zone
- If reduced opacity needed, pre-bake via RGBA — never use CSS opacity

WHAT IS CLAUDE CODE?

An AI coding agent you can structure

An AI agent that reads, writes, and runs code in your terminal. Four things determine how well it works:

Context — what it knows about your project

Tools — what it can do (read, write, search, run)

Model — how it thinks and reasons

Prompt — how you direct it (CLAUDE.md, commands)

Built-in: /help, /clear, /compact, /model, /config, /permissions

This template structures all four pillars. The intersection is where quality happens.

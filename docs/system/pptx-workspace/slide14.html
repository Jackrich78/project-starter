THE COMPOUNDING ADVANTAGE

The longer you use it, the better the context

Feature Docs

Every /explore, /blueprint, /build, and /commit writes to docs/features/FEAT-XXX/. The PRD, plan, handover, and QA report persist across sessions. Feature 12 benefits from everything written in features 1–11.

Skills

After each feature, /retro extracts patterns into .claude/skills/. A debugging approach from FEAT-003. A validation pattern from FEAT-007. Each skill is auto-imported when relevant commands run.

Session Continuity

/handover captures state in under 1,500 tokens — what was done, decisions made, what's next. /prime loads it at session start. Each merge prunes stale context. Handovers get sharper over time, not longer.

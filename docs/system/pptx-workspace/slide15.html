GETTING STARTED

Up and running in 15 minutes

Step 1 — Clone the template

git clone https://github.com/jackrich78/project-starter.git my-project

Or use "Use this template" on GitHub to create a copy under your account

Step 2 — Install and test

npm install && npm test

All 10 scaffold tests should pass before you continue

Step 3 — Open in Claude Code

claude .

Claude Code reads CLAUDE.md automatically and loads project context

Step 4 — Run /setup to configure your project

/setup docs/research/my-project-notes.md

Configures PROJECT.md, CLAUDE.md, and generates initial docs from your source material

Step 5 — Start your first feature

/explore your first feature idea

Explore interviews you and creates the first PRD — the pipeline begins

Run npm test first — all 10 scaffold tests should pass before you start.
